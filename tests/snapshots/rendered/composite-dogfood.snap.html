Frequently asked questions

  • Install AWT from the WordPress admin search. The free theme and blocks plugin both live on WordPress.org. Activate the theme, then activate the blocks plugin to get every Carbon component as a real block.

  • AWT runs alongside WooCommerce without breaking. WooCommerce's own templates render with WC default styling at Stage 1. Full Carbon-styled WooCommerce templates are part of AWT Premium.

  • Yes. The Site Editor's palette panel lets you override any Carbon token. Contrast is checked against both light and dark theme scopes so you cannot accidentally ship an inaccessible palette.
